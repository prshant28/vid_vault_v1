Build a **production-ready, AI-powered web application** called **"VidVault AI"** — a next-generation "Second Brain for Videos".

The app must combine:

* Video bookmarking (like YouTube)
* Knowledge management (like Notion)
* AI assistant (like ChatGPT)

---

# 🎯 CORE VISION

Users should be able to:

1. Paste any video URL
2. Instantly save it as a smart card
3. Organize it into folders/tags
4. Use AI to transform video into knowledge (notes, PPT, MCQs, etc.)
5. Search and retrieve insights instantly

---

# 🔄 CORE FLOWS

## Flow 1: Add Video

* User pastes URL
* Backend fetches metadata
* Save to DB
* Show as card

## Flow 2: AI Generation

* User clicks “AI”
* Send video data → OpenAI
* Store output
* Display in sidebar

## Flow 3: Organization

* Drag card → folder
* Add tags
* Filter/search
